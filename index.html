Минимальные требования¶

OC: Windows 7 и выше

Процессор: не менее Intel Pentium G4650

RAM: 2GB

Библиотеки и требуемые программы: .NET Framework 4.7, Microsoft Word

Установка¶

1. Скачать программу по ссылке: ТЫК

2. Распаковать архив любым архиватором, в любое место

3. Запустить файл "lada.exe", который лежит в корне каталога

4. Готово!

Использование¶

Регистрация и вход

1. Нажать кнопку "Зарегистрироваться"

2. Ввести логин

3. Вести пароль

4. Повторить пароль

5. Нажать кнопку "Регистрация"

6. Готово! Вы успешно зарегистрировали аккаунт!

7. Ввести логин

8. Вести пароль

9. Нажать кнопку "Войти"

10. Готово! Вы успешно вошли в свой аккаунт!

Добавление материалов

1. Нажать кнопку "Другие формы"

2. Нажать кнопку "База материалов"

3. Открылась форма "База материалов"

4. Нажать на кнопку "Добавить заказ"

5. Ввести соотвествующие данные

6. Нажать кнопку "Сохранить данные"

7. Готово! Вы успешно добавили материал

8. Закрыть форму "База материалов"

Перенос данных в Microsoft Word

1. Нажать кнопку "Файл"

2. Нажать кнопку "Экспорт в Word"

3. В окрывшемся окне нажать кнопку "OK"

4. Экспортированный файл откроется автоматически

P.S. Экспортированный файл хранится там же, где и сама программа.
